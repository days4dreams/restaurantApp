JSC Unit 9 Project

In this project, you’ll practice building out a base landing page using HTML and CSS, with the intent of adding JavaScript later. You can use this landing page for your final project if you choose. All of your work this week will be built in Sublime Text and submitted via GitHub.

Let’s get started!

Imagine a small restaurant has hired you to build a simple website that accepts online reservations. Don’t worry about the functionality of the reservations just yet — your goal is to practice setting up a web page to which you can add JavaScript.

The restaurant has provided you with the following elements the website must include:
- The name of the restaurant
- A large image of the restaurant
- A section for reservations that includes:
- A form to make reservations
- A table to view existing reservations
- A section to view the map of its location
- A customer review

The images you’ll need for the landing page can be found here:

  2. Tom's Restaurant

Step 3
In the “Reservations” section, add HTML to the “Make a Reservation” column to build a form that includes:
A text input field for the person’s name
A text input field for the day
A button to confirm the reservation

If you need a refresher on forms, use the code below as a guide: 

<form>
    <input type="text" placeholder="Placeholder Text">
    <button>Confirm Reservation</button>
</form>

Step 4
Next, expand the “Existing Reservations” column in the “Reservations” section to show the data the customer will enter. Don’t worry about JavaScript functionality for now. Be sure the existing reservations are stored in a table with the following elements:

Two table headers, labeled “Name” and “Day”
A body

If you need a refresher on formatting a table, use this code as a guide:

<table>
<thead>
<th>Column Header 1</th>
<th>Column Header 2</th>
</thead>
<tbody> [ form data will publish here ] </tbody>
</table>
